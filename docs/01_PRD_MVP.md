# 01 PRD MVP
Last updated: 2026-01-22

This is a lightweight PRD stub that points to the current execution plan and scope docs. It defines the MVP behaviors that must ship in Week 2.

## MVP Definition
- Today view shows a daily pairing (literature + verse) with calm reading flow.
- Detail view expands the pairing with verse text and short rationale.
- Save/unsave enables a personal list of readings.
- Quiet invite delivery sends daily content with approved-only pairing.
