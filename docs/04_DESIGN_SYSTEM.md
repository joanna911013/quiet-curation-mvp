# 04 Design System
Last updated: 2026-01-22

Design system summary for the MVP. This doc points to the canonical design principles, typography, components, and pairing specs.

## Summary
- Reading-first layouts with quiet, minimal chrome.
- System UI typography with calm tone and wide spacing.
- Pairing block is secondary; absence is quiet, not broken.
