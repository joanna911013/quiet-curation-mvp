# Landing Lock (Week 5 Day 1)
Date: 2026-02-09

## Landing IA (6 Sections)
- Hero — Purpose: state the promise and invite a 3‑minute screen‑loop break. Key elements: H1, sub, microcopy, primary CTA, secondary link, screen‑loop line.
- How — Purpose: explain the 3‑minute flow in plain steps. Key elements: 3 steps, time commitment note.
- Sample — Purpose: show one pairing preview to set expectations. Key elements: sample card, attribution line.
- Trust — Purpose: reduce anxiety about cadence and privacy. Key elements: daily time, unsubscribe, no feed/ads.
- FAQ — Purpose: answer common doubts quickly. Key elements: 3–5 short Q/A items.
- Final CTA — Purpose: last chance to subscribe. Key elements: CTA button + quiet reassurance.

## Hero Copy Lock
- Headline: "Break the screen‑loop in 3 minutes."
- Sub: "One literature × verse pairing, delivered by email. No feed. No noise."
- Microcopy: "screen‑loop → 3‑minute dopamine detox → quiet invite (email)"
- Primary CTA: "Join the quiet invite"
- Secondary link: "See a sample pairing"

## Tone Rules (Quiet / Non‑Hype)
Forbidden (avoid):
- life‑changing
- viral
- guaranteed
- hustle
- revolutionary

Recommended (prefer):
- quiet
- calm
- gentle
- small daily ritual
- take a breath

## Responsive Rules (Core)
- Mobile fold must show the primary CTA without scrolling.
- H1 line breaks: max 2 lines (use width clamp if needed).
- Body line length target: 32–40 characters (~`max-w-[38ch]`).
- Button tap target: minimum height 44px.

## Material Reference (Direction)
- Follow Material-inspired surfaces and elevation (soft container backgrounds, light shadow).
- Use clear hierarchy with filled primary button + text secondary actions.
